Let 바인딩

"let 바인딩"은 다른 프로그래밍 언어로 치면 "변수 선언"이라 할 수 있습니다. let은 이름에 값을 묶어줍니다. 이 이름들은 let 바인딩 뒤에 사용될 수 있습니다.

Let 바인딩은 일반적인 JavaScript 변수 선언문으로 컴파일됩니다. 우리는 다른 JavaScript 라이브러리들과 연계 및 쉬운 디버깅을 위해 컴파일 과정에서 let 바인딩의 이름들을 최대한 같게 유지합니다.

Reason
Output
let greeting = "hello!";
let score = 10;
let newScore = 10 + score;
var greeting = "hello!";
var score = 10;
var newScore = 20;

블럭 스코프

{}를 통해 바인딩의 범위를 설정 가능합니다.

Reason
Output
let message = {
  let part1 = "hello";
  let part2 = "world";
  part1 ++ " " ++ part2
};
/* `part1`과 `part2`는 여기서 참조 불가능합니다! */
var message = "hello world";

암시적으로 {} 블록의 마지막 줄의 값이 반환됩니다.

디자인 결정

Reason의 if, while, 그리고 함수 구문 모두 위와 똑같은 블럭 범위 문법을 사용합니다. 이 코드는 특별한 if의 {} 범위 문법으로 동작하는 것이 아니며, 단순히 위에서 본 것과 같은 문법으로 동작하는 것입니다.

Reason
if (displayGreeting) {
  let message = "Enjoying the docs so far?";
  print_endline(message)
};
/* `message`는 여기서 참조 불가능합니다! */

바인딩은 불변입니다

Reason의 let 바인딩은 "불변", 즉 이름에 값이 묶인 후 값이 변경될 수 없습니다. 이 성질은 Reason의 타입 시스템이 더욱 많은 것을 최적화하고 추론할 수 있도록 할 수 있게 합니다. (결과적으로, 이는 당신을 더 편하게 합니다)

Binding Shadowing

위의 제한은 처음에는 실용적이지 않게 들릴 수 있습니다. 만약 그렇다면 어떻게 값을 바꿀 수 있을까요? 주로, 두 방법이 사용됩니다.

첫 번째 방법은 당신이 원하는 것은 변수의 값을 변경하는 것이 많은 경우에 아니라는 것을 깨닫는 것입니다. 예시로, JavaScript에서는

JS
var result = 10;
result = Math.random() + result;
result = Math.floor(result);

여기서 정말로 하는 것은 단지 중간 과정들을 저장하는 것입니다. 당신은 result 변수를 변경할 필요가 하나도 없었습니다! 이 코드는 이렇게 다시 쓸 수 있습니다.

JS
var result1 = 10;
var result2 = Math.random() + result1;
var result3 = Math.floor(result2);

Reason에서도 이 방식은 당연히 동작합니다.

Reason
Output
let result1 = 10.;
let result2 = Js.Math.random() +. result1;
let result3 = Js.Math.floor_float(result2);
var result1 = 10;
var result2 = Math.random() + 10;
var result3 = Math.floor(result2);

어쨌든, Reason에서는, 우리는 let 바인딩 "가리기"라는 문법을 지원합니다. 따라서 당신은 위 코드를 이렇게도 쓸 수 있습니다.

Reason
Output
let result = 10.;
let result = Js.Math.random() +. result;
let result = Js.Math.floor_float(result);
var result = Math.random() + 10;
var result$1 = Math.floor(result);

사실은, 심지어 이 코드도 괜찮습니다.

Reason
Output
let result = "hello";
Js.log(result); // "hello"가 출력됩니다
let result = 1;
Js.log(result); // 1이 출력됩니다
console.log("hello");
console.log(1);

각 이름이 참조하는 값은 어느 쪽이든 위쪽으로 가장 가까이에 있는 것입니다. 변수의 변경은 없었죠!

실제 뮤테이션이 필요하다면, (예를들어 값을 전달하고 많은 코드 조각으로 수정한다면) 좀 더 무거운 뮤테이션 기능을 제공합니다.
